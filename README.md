# Customer-support-dataset-with-multi-itent-annotations-for-conservational-AI
📌 Project Overview
------------------------------------------------------------------------------------------------------------------------------------------------------------------
This project presents a structured and annotated customer support dataset designed for training conversational AI systems to handle multi-intent user queries effectively.

The dataset captures real-world customer interactions where a single query may contain multiple intents, enabling the development of more intelligent, context-aware, and robust dialogue systems.
